# 规格文档 - 输出数据规格定义

## 1. 输出文件

### 1.1 文件列表

| 文件名 | 格式 | 说明 |
|--------|------|------|
| `output/financial_data.xlsx` | Excel | 主输出文件，包含全部数据 |
| `output/financial_data.csv` | CSV | 便于程序处理的平面文件 |

### 1.2 输出目录结构

```
output/
├── financial_data.xlsx
└── financial_data.csv
```

## 2. 数据结构

### 2.1 字段定义

| 列名 | 类型 | 说明 | 示例 |
|------|------|------|------|
| stock_code | string | 股票代码（6位） | 601012 |
| stock_name | string | 公司简称 | 隆基绿能 |
| report_date | date | 报告期末日期 | 2024-03-31 |
| quarter | string | 季度标签 | 2024Q1 |
| net_profit | float | 净利润（元） | 1234567890.12 |
| total_assets | float | 总资产（元） | 98765432100.00 |
| total_liabilities | float | 总负债（元） | 56789012300.00 |
| revenue | float | 营业收入（元） | 23456789000.00 |

### 2.2 数据排序
- 主排序：stock_code 升序
- 次排序：report_date 升序

### 2.3 缺失值处理
- 公司尚未上市的季度：不生成对应行
- API返回空值：字段值为空（NaN/空白）

## 4. 数据量预估

- 30家公司 × 最多27个季度 = 最多810行
- 实际行数会少于810行（部分公司上市较晚）
- 每行8个字段

## 5. 编码与格式

- 文件编码：UTF-8 (CSV带BOM以兼容Excel)
- CSV分隔符：逗号
- Excel格式：xlsx (openpyxl)
- 数值精度：保留原始精度（小数点后2位）
- 日期格式：YYYY-MM-DD
